# Flick App Icon

Drop one final 1024 x 1024 PNG into this folder with this exact name:

- `AppIcon-1024.png`

When the app builds, Xcode runs a small local script that generates the smaller macOS icon sizes:

- `AppIcon-16.png`
- `AppIcon-32.png`
- `AppIcon-64.png`
- `AppIcon-128.png`
- `AppIcon-256.png`
- `AppIcon-512.png`

Use a square PNG. A 1024 x 1024 image is best.
